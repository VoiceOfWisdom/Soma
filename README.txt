<English>
The Soma assembly file is Soma.Core.dll.
It is placed in following folder.

.\bin\Release

Soma depends following assemblies.
1)FSharp.Core.dll
2)FSharp.PowerPack.dll
3)mscorlib.dll
4)System.dll
5)System.Core.dll
6)System.Data.dll
7)System.Numerics.dll
8)System.Transactions.dll

Download FSharp.Core.dll from Microsoft Research.
http://research.microsoft.com/en-us/um/cambridge/projects/fsharp/release.aspx

FSharp.PowerPack.dll is placed lib folder.

Others are included in .NET Framework 4.
