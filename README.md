Soma - F# Sql Oriented MApping framework
=====================================

Soma is an O/R mapping framework developed in F#. 

Supported programming languages and RDBMS are followings:

> languages

* F# 2.0
* C# 4.0
* Visual Basic 2010
* PowerShell

> RDBMS

* Microsoft SQL Server 2008
* Microsoft SQL Server Compact 4.0
* Oracle Database 11g
* MySQL 5.x.
* SQLite 3
